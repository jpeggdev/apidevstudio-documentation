---
title: Settings Reference
description: Complete reference for all application settings
---

Open Settings with `Ctrl+,` or click the gear icon in the sidebar.

## Appearance

| Setting | Description | Default |
|---------|-------------|---------|
| **Theme** | Dark, Light, or System | Dark |
| **Animations** | Enable/disable UI animations | Enabled |

## Server

| Setting | Description | Default |
|---------|-------------|---------|
| **Default port** | Port assigned to new projects | 3001 |

## Startup

| Setting | Description | Default |
|---------|-------------|---------|
| **Close to system tray** | Minimize to tray instead of quitting when closing the window | Off |
| **Start minimized** | Launch the app minimized to the system tray | Off |
| **Launch on system startup** | Automatically start the app when you log in | Off |

## Updates

| Setting | Description | Default |
|---------|-------------|---------|
| **Changelog** | View release notes for current and past versions | - |
| **Auto-check for updates** | Periodically check for new versions | On |
| **Auto-download updates** | Download updates in the background | Off |
| **Auto-install updates** | Install updates automatically when downloaded | Off |

## License

| Setting | Description |
|---------|-------------|
| **License status** | Shows Active (Pro) or Free |
| **License email** | Email associated with your Pro license |
| **Machine ID** | Unique identifier for this device |
| **Activate License** | Enter a license key or use email activation |
| **Deactivate License** | Release the license for use on another device |

See [Licensing & Pro Features](../features/licensing) for more details.

## Developer (Hidden)

These settings appear only when Developer mode is enabled (press `Ctrl+Shift+D` three times on the Settings page).

| Setting | Description |
|---------|-------------|
| **Developer mode** | Shows current developer mode status |
| **View Error Reports** | Open the Error Report Manager |

See [Error Reporting](../features/error-reporting) for more details.
